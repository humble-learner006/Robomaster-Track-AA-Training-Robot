# 关于机械组未来发展规划
整理于蔡礼珂和李顺的对话，未标明李顺观点的部分是蔡礼珂的观点

我的动机是物料管理太混乱，机械的备件，工具，每次都需要长时间寻找。所以想增加车组负责人的机制

但是在收 ml 之前，给每个车组分一个机械组负责人  
你觉得负责人这个方法怎么样？  
让他们去管理每个车组/机构的物资，把物料管理的压力分散出去  
我觉得我们可以慢慢培养。
先把责任给出去，再提升能力

## 李顺的反对意见
1. 单调工作内容：那如果这样的话那个人（负责人）不就相当于只做一个兵种了
2. 有空闲情况产生，如英雄装完了干英雄的不彻底歇着了
3. 成员实力不足：有那么多能靠得住的人吗
4. 成员规模不足：就最简单的工程和无人机就很难找到人

# 对于机械组成员比赛生涯的预想
大一招进来让学长带着在固定的车组熟悉机械
等到大二的时候当负责人
龙大特色是大三队员可能交换/转到其他组，所以我对于机械这边的成员生涯都按照两年规划

我的意见：当负责人其实平时挺闲的，真没太多维护任务，其他时间完全可以为转组做准备。选组之前和他们说明当下情况和利弊吧  
李顺的意见:那机械剩下任务重组别会有意见

# 对于当前情况的认识
现在我们是重复刻队伍，在过去三年内快速拓展出全兵种机械  
但是在未来不会有大规模造新车的需求，所以我认为机械的责任会往维护现有车辆发展
